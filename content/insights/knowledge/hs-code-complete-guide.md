**HS Code（Harmonized System Code）**，即协调制度编码。无论你的货物发往哪个国家，海关都需要通过这串数字来确定它是“什么”，从而决定征收多少关税。

## 1. HS Code 的结构（以中国为例）

标准的 HS Code 通常由 6 到 10 位数字组成：
* **前 2 位：** 章（Chapter），代表商品大类。
* **前 4 位：** 目（Heading），进一步细化类别。
* **前 6 位：** 全球通用的子目（Sub-heading）。
* **后几位：** 各国根据自身贸易、统计需求增设的本国子目。

---

## 2. 为什么 HS Code 如此重要？

* **决定税率：** 选错编码可能导致从 0% 关税变成 25% 关税。
* **监管要求：** 某些编码对应的商品需要出口许可证或特定商检（如植检证、3C认证）。
* **合规风险：** 故意错报编码被视为“逃税”或“瞒报”，可能导致货物被扣押或没收。

---

## 3. 常见归类错误避坑点

1. **按用途归类而非材质：** 比如一个塑料制手机壳，应归类在“塑料制品”下，而非“通讯设备”下。
2. **忽视“申报要素”：** 很多编码要求注明规格、型号、品牌。
3. **混淆零件与成品：** 机器的零件和机器本身往往对应不同的编码和税率。
